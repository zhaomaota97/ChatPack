# ChatPack - 实现指南

## 概述

本文档补充说明实现项目时需要注意的关键细节、边界情况和技术决策。

---

## 1. 初始化数据

### 1.1 新用户注册
新用户注册后自动获得：
- **初始卡包**: 3个"卡包"（普通卡包，配置为 60/30/8/2 的混合概率）
- **邀请码**: 自动生成8位唯一邀请码

### 1.2 系统预设聊天室
系统初始化时应创建以下聊天室（建议配置）：

```sql
-- 示例初始化数据
INSERT INTO chat_rooms (name, description) VALUES
  ('🌱 小学乐园', '适合小学水平的聊天室'),
  ('🌿 初中世界', '初中水平单词交流'),
  ('🌳 高中殿堂', '高中水平单词交流'),
  ('🎓 四级广场', '四级单词交流广场'),
  ('🏆 六级天地', '六级单词挑战'),
  ('👑 考研领域', '考研单词终极挑战');

-- 关联单词书（示例）
-- 小学乐园 - 只关联小学单词书
-- 初中世界 - 关联小学+初中
-- 高中殿堂 - 关联小学+初中+高中
-- 等等...
```

### 1.3 默认卡包配置
建议预设以下卡包：

| 卡包名称 | 配置 | 说明 |
|---------|------|------|
| 卡包 | `{"COMMON":60,"RARE":30,"EPIC":8,"LEGENDARY":2}` | 新手默认卡包 |
| 稀有卡包 | `{"COMMON":0,"RARE":100,"EPIC":0,"LEGENDARY":0}` | 100%稀有 |
| 史诗卡包 | `{"COMMON":0,"RARE":0,"EPIC":100,"LEGENDARY":0}` | 100%史诗 |
| 传说卡包 | `{"COMMON":0,"RARE":0,"EPIC":0,"LEGENDARY":100}` | 100%传说 |
