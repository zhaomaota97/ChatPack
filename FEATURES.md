# ChatPack - 功能说明

## 项目简介

ChatPack（单词十连抽）是一个英语单词学习游戏平台。通过「开卡包收集单词」和「分级聊天室交流」的游戏化方式学习英语。

## 核心功能

### 1. 🎴 卡包系统

**收集单词如同收集卡牌**

- **卡包**: 打开卡包获得单词
- **稀有度机制**: 单词分为普通、稀有、史诗、传说四个稀有度
- **开包惊喜**: 每次开包都能获得新单词，不会重复
- **精美展示**: 开包动画和稀有度视觉效果

**卡包类型:**
- 普通卡包 (5张) - 包含所有稀有度卡牌
- 稀有卡包 (5张) - 只开出稀有稀有度单词
- 史诗卡包 (5张) - 只开出史诗稀有度单词
- 传说卡包 (5张) - 只开出传说稀有度单词

### 2. 💬 分级聊天室

每个人都只能用自己拥有的单词以及符号、阿拉伯数字来发言

### 3. 📚 单词库存

**你的单词收藏馆**

- **库存管理**: 查看所有收集的单词
- **搜索筛选**: 按稀有度、等级、收藏状态筛选
- **收藏功能**: 标记常用或想复习的单词
- **详细信息**: 查看单词释义、音标、例句
- **统计数据**: 收集进度、稀有度分布

### 4. 👤 用户系统

**个性化学习体验**

- **账号管理**: 注册、登录、资料编辑
- **邀请机制**: 每个用户有独特邀请码
- **成就记录**: 累计开包数、单词收集数
- **头像昵称**: 自定义个人形象

## 游戏机制

### 单词稀有度系统

| 稀有度 | 颜色 | 开包概率 | 特点 |
|--------|------|---------|------|
| COMMON | 白色 | 35% | 常见单词，基础词汇 |
| RARE | 蓝色 | 30% | 较少见单词 |
| EPIC | 紫色 | 25% | 高级词汇 |
| LEGENDARY | 金色 | 10% | 稀有高级词汇 |

**注意:** 稀有度是全单词设计的，卡包

### 环境等级系统

聊天室的环境等级决定了可用单词范围：

```
小学环境  →  只能用小学单词
初中环境  →  小学 + 初中
高中环境  →  小学 + 初中 + 高中
四级环境  →  四级及以下所有
六级环境  →  六级及以下所有
考研环境  →  所有单词
```

这种设计鼓励用户：
1. 在低级环境中熟练使用基础单词
2. 逐步挑战高级环境
3. 收集更多单词以解锁更多表达方式

## 学习路径

### 新手阶段
1. 注册账号
2. 获得新手卡包
3. 开包收集首批单词
4. 进入小学乐园练习

### 进阶阶段
1. 收集更多卡包
2. 扩充单词库存
3. 尝试更高级聊天室
4. 与其他玩家互动

### 高级阶段
1. 收集稀有单词
2. 挑战考研领域
3. 帮助新手玩家
4. 完成词汇收集

## 社交互动

### 实时聊天
- 看到其他用户在线
- 实时收发消息
- 回复特定消息
- 观察他人用词
- **送花互动**: 给喜欢的发言送花，可取消

### 送花系统
- 给任何消息送花，消息鲜花数+1
- 发言人总鲜花数累计增加
- 送花不消耗自己的鲜花（免费无限）
- 可以取消送花

### 学习氛围
- 通过观察学习: 看到单词的实际使用场景
- 良性竞争: 尝试使用更高级或更贴切的单词
- 互相启发: 发现新的单词用法

## 管理功能

### 管理员后台

**单词管理:**
- 添加、编辑、删除单词
- 批量导入单词数据
- 管理单词所属单词书
- 调整单词稀有度

**单词书管理:**
- 查看单词书内容
- 添加/移除单词
- 启用/禁用单词书

**聊天室管理:**
- 创建/编辑/删除聊天室
- 自定义聊天室名称和描述
- 配置聊天室可用单词书等级
- 启用/禁用聊天室

**用户管理:**
- 查看用户列表
- 封禁/解封用户
- 赠送卡包
- 查看用户详细统计

**卡包管理:**
- 创建、编辑、删除卡包
- 配置卡包参数（名称、描述、数量）
- 设置卡包稀有度类型
- 启用/禁用卡包

**数据统计仪表盘:**
- 用户统计：总数、活跃用户、新增用户
- 单词统计：总数、稀有度分布、单词书分布
- 卡包统计：开包数、库存数
- 消息和鲜花统计
- 可视化图表展示

**系统设置:**
- 全局配置管理
- 系统参数调整
